# Пистолет Ярыгина (ПЯ)

## Разработка

Установка зависимостей:

```bash
npm ci
```

Запуск [Rollup](https://rollupjs.org):

```bash
npm run dev
```

Проект будет запущен на [localhost:5000](http://localhost:5000).

## Сборка

Для сборки production-версии запустить

```bash
npm run build
```

Поставляется содержимое папки `public`.
